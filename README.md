# Stellar Notebook - Mockup UX Design

Mockup notatnika z funkcjami AI zaprojektowany dla studentów.

## Funkcje

- **Rich Text Editor** - edytor tekstu z formatowaniem
- **Graf powiązań** - wizualizacja połączeń między notatkami
- **Tryby pracy**:
  - Tryb Syntezy - łączenie notatek
  - Tryb Rozmowy - chatbot do pytań o notatki
  - Tryb Nauki - chatbot zadający pytania
- **Funkcje AI**:
  - Szybka korekta
  - Zaawansowana korekta (formatowanie)
  - Rozszerzanie treści
  - Generowanie pytań do nauki
  - Niestandardowy prompt

## Design

- Sci-fi, czarnofioletowy motyw
- Animowane gwiazdki w tle
- Efekty holograficzne
- Futurystyczna typografia

## Instalacja

```bash
npm install
npm run dev
```
